# Technology Stack

## Core

- **Runtime & Package Manager**: [Bun](https://bun.sh/) - Used for lightning-fast execution, dependency management, and type support.
- **Language**: [TypeScript](https://www.typescriptlang.org/) - Ensuring type safety and better developer experience.
- **Framework**: [Svelte 5](https://svelte.dev/) & [SvelteKit](https://kit.svelte.dev/) - Modern, compiler-first framework for high-performance web applications.

## Frontend & UI

- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) - Utility-first CSS framework for rapid UI development.
- **UI Components**: [Shadcn-Svelte](https://shadcn-svelte.com/) - Accessible and customizable UI components built with Svelte.
- **Aesthetics**: [Retro UI / Neo-Brutalism](https://retroui.dev/) - Bold, high-contrast design language.

## Infrastructure

- **Containerization**: [Docker](https://www.docker.com/) - Consistent development and deployment environments.
- **Hosting**: [Google Cloud Run](https://cloud.google.com/run) - Serverless platform for deploying containerized applications.

## Persistence

- **Local Storage**: [Web localStorage API](https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage) - For local-first persistence of game data and high scores.
